\# Risk-Aware Pre-warming for Serverless Systems



This repository contains the implementation of a proactive, risk-aware pre-warming framework

to mitigate cold start latency in serverless computing.



\## Overview

The project explores:

\- Distributional workload forecasting using Quantile Temporal Convolutional Networks (TCN)

\- Extreme event modeling using Extreme Value Theory (EVT)

\- Cost-aware decision making using Conditional Value at Risk (CVaR)



The framework is evaluated using large-scale public Function-as-a-Service (FaaS) traces.
